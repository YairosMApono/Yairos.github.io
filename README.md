## Spielen

- **Online:** [GitHub Pages](https://yairosmapono.github.io/Yairos.github.io/)
- **Lokal:** `index.html` in einem Browser öffnen

## Projektstruktur

```
├── index.html          # Einstiegspunkt
├── css/style.css       # Design System & Layout
├── js/
│   ├── config.js       # Spielkonstanten
│   ├── game.js         # Spiel-Logik
│   └── ui.js           # UI-Rendering
└── docs/
    ├── README.md       # Diese Datei
    ├── ARCHITECTURE.md # Architektur
    ├── GAMEPLAY.md     # Spielmechaniken
    └── DESIGN.md      # Design-System
```

## Technologie

- **HTML5, CSS3, Vanilla JavaScript** – Keine Frameworks
- **localStorage** – Persistenz
- **GitHub Pages** – Hosting
