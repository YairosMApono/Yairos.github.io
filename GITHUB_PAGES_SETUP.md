# GitHub Pages aktivieren

Falls die Seite noch 404 anzeigt, **GitHub Pages einmalig einrichten**:

1. Öffne: **https://github.com/YairosMApono/Yairos.github.io/settings/pages**
2. Unter **"Build and deployment"** → **"Source"** wähle: **"GitHub Actions"**
3. Speichern – der Workflow deployt automatisch bei jedem Push

Nach 1–2 Minuten sollte die Seite unter **https://yairosmapono.github.io/** erreichbar sein.
