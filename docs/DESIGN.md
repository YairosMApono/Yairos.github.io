# Empire Builder – Design-System

## Farben

| Variable | Hex/RGBA | Verwendung |
|----------|----------|------------|
| `--gold` | #c9a227 | Akzente, aktive Elemente |
| `--gold-light` | #e8d48b | Hervorgehobener Text |
| `--gold-dim` | rgba(201,162,39,0.15) | Hover-Hintergründe |
| `--parchment` | #f4e4bc | Haupttext |
| `--forest` | #1a472a | Hintergrund, Theme |
| `--surface` | rgba(26,35,26,0.95) | Karten, Container |
| `--success` | #8fbc8f | Positive Werte |
| `--danger` | #e07a7a | Negative Werte, Fehler |

## Typografie

- **Überschriften:** Cinzel (serif)
- **Fließtext:** Crimson Text (serif)

## Spacing & Radius

- **Radius:** 24px (XL), 20px (LG), 16px (MD), 12px (SM), 8px (XS)
- **Spacing:** 0.5rem, 1rem, 1.5rem, 2rem (8px-Grid)

## Schatten

- `--shadow-soft`: Leichte Tiefe
- `--shadow-medium`: Modale, Karten
- `--shadow-glow`: Goldener Glow bei Hover/Aktiv

## Responsive

- **Desktop:** Volle Breite bis 1400px
- **Tablet:** < 600px – 2-Spalten-Layout, kompaktere Ressourcen
- **Mobile:** < 380px – Minimale Abstände
